## 🛠️ Tecnologias Utilizadas

### Front-end
- **HTML5** - Estrutura semântica e acessível
- **CSS3** - Design moderno com variáveis CSS e animações
- **JavaScript ES6+** - Funcionalidades dinâmicas e interativas
- **Font Awesome 6** - Ícones vetoriais profissionais

### Back-end & Banco de Dados
- **Supabase** - Backend como serviço (alternativa open-source ao Firebase)
  - Autenticação
  - Banco de dados PostgreSQL
  - APIs em tempo real
  - Storage para arquivos

### Hospedagem
- **GitHub Pages** - Hospedagem gratuita e rápida
- **SSL Automático** - Segurança garantida
